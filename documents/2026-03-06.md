# 회의록 (2026-03-06)

## 1. 회의 개요
- 프로젝트명: 소재부품 수명 예측 모델 개발
- 회의 목적: 데이터/문헌 분석 결과 공유 및 초기 모델링 방향 수립

## 2. 금주 진행 사항

### 2.1 문헌 조사 및 기술 검토
- Bhadeshia 교수 및 Tancret 등 관련 연구 논문 3편을 분석함.
- 초합금 및 내열강의 크립 수명 예측에 활용 가능한 알고리즘 특성을 검토함.
- 물리적 법칙을 위배하지 않는 범위에서 데이터 기반 예측을 수행하는 방향을 채택함.

### 2.2 데이터 구조 분석
- 제공 데이터 파일의 구조를 확인함.
- 총 31개 컬럼을 점검함.
  - 타깃: 수명(Lifetime)
  - 입력: 응력, 온도, 18종 원소 함량, 열처리 조건 등
- 물리 단위(K, MPa, wt%)를 확인하고 입력 규격 초안을 수립함.

### 2.3 모델링 초기 방향
- 검토 모델: XGBoost, MLP, Random Forest
- 비선형 관계를 반영하면서도 물리적 해석 가능성을 유지하는 방향으로 모델 후보군을 구성함.

## 3. 핵심 이슈 및 리스크
- Lifetime의 수치 범위가 매우 넓어 학습 수렴 저하 및 오차 확대 가능성이 있음.
- 데이터셋 규모가 작아 과적합과 변수 편향 위험이 큼.
- 소규모 데이터 특성상 모델 복잡도 관리와 검증 체계가 중요함.

## 4. 주요 결정 사항
- 전처리 파이프라인 설계를 우선 수행함.
- 타깃 분포 편향 완화를 위한 로그 변환 적용 가능성을 중점 검토함.
- 소규모 데이터에 적합한 모델 선택 기준을 차주 회의에서 구체화함.

## 5. 차주 계획
- 로그 변환 수식의 적절성을 데이터 분포 관점에서 검증함.
- 데이터 누수 방지 원칙을 포함한 학습/평가 분리 전략을 설계함.
- 후보 모델별 장단점 및 적용 우선순위를 확정함.

## 6. 기대 산출물
- 전처리 파이프라인 초안
- 데이터 입력 규격 정리본
- 모델 후보군 비교 메모(적용 조건, 리스크, 기대 성능)

## 7. 결론
- 본 회의는 데이터 특성과 리스크를 조기에 식별하고, 전처리 중심의 실무형 개발 방향을 확정한 단계임.
- 다음 단계의 핵심은 분포 보정 및 검증 체계 고도화를 통한 모델 신뢰성 확보임.
