# 회의록 (2026-03-16)

## 1. 회의 개요
- 프로젝트명: 소재부품 수명 예측 모델 개발
- 회의 목적: 전처리 구현 결과 점검 및 모델/평가 체계 확정

## 2. 금주 진행 사항

### 2.1 전처리 파이프라인 구축
- 타깃 변수(Lifetime)의 넓은 수치 범위 및 편향 분포 문제를 완화하기 위해 로그 변환을 적용함.
- 독립 변수(합금 성분 wt%, 응력, 온도, 열처리 조건)의 스케일 차이를 보정하기 위해 StandardScaler 기반 정규화를 수행함.

### 2.2 데이터 분할 및 검증 준비
- 데이터 누수를 방지하는 원칙에 따라 Train/Test 분리를 완료함.
- 평가의 객관성과 재현성을 확보하기 위한 기본 검증 절차를 정리함.

### 2.3 모델 비교 검토
- 후보 모델의 데이터 적합성을 비교함.
  - MLP: 파라미터 수가 많아 소규모 데이터에서 과적합 위험이 큼
  - XGBoost, Random Forest: 비선형 관계 대응력과 일반화 측면에서 상대적으로 유리
- 현재 데이터 조건에서는 트리 기반 앙상블 중심 접근이 타당하다는 중간 결론을 도출함.

## 3. 주요 의사결정 사항
- 전처리 기준안으로 로그 변환 + 표준화 조합을 유지함.
- 우선 검증 모델을 XGBoost 및 Random Forest로 설정함.
- 성능 평가는 RMSE, R²를 기본 지표로 사용하고 산점도(실측 vs 예측) 분석을 병행함.

## 4. 리스크 및 대응
- 리스크 1: 소규모 데이터 기반 과적합
  - 대응: 모델 복잡도 제어, 지표 기반 비교, 오차 패턴 분석 강화
- 리스크 2: 로그 역변환 구간에서의 해석 오차
  - 대응: 역변환 후 오차 분포를 별도 점검하고 해석 기준을 명확화
- 리스크 3: 특정 조건 구간(응력/온도) 편향 가능성
  - 대응: 구간별 오차를 분리 분석하여 취약 구간을 식별

## 5. 차주 계획
- XGBoost/Random Forest 학습 및 하이퍼파라미터 조정 수행
- RMSE, R² 기반 모델 성능 비교표 작성
- 실측-예측 산점도 기반 오차 경향 분석
- 1차 기준 모델 선정 및 후속 개선 과제 정의

## 6. 기대 산출물
- 모델 성능 비교표(RMSE, R²)
- 실측 대비 예측 산점도 및 오차 분석 메모
- 1차 기준 모델 선정 근거 문서

## 7. 결론
- 전처리 및 데이터 분할 체계가 안정적으로 구축되었고, 모델 비교를 위한 준비가 완료됨.
- 다음 단계는 정량 평가와 시각화 분석을 통해 실제 적용 가능한 기준 모델을 확정하는 것임.
