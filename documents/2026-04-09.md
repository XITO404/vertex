# 회의록 (2026-04-09)

## 1. 회의 개요
- 프로젝트명: 소재부품 수명 예측 모델 개발
- 회의 목적: 데이터 소스 통합 진행 현황 점검 및 제품군 기반 검증 전략 고도화 방향 확정

## 2. 금주 진행 사항

### 2.1 데이터 가용성 확대를 위한 소스 통합
- 확보된 실험 데이터셋과 웹 크롤링 기반 외부 데이터의 병합을 위해 스키마 매핑 작업을 수행함.
- 데이터 부족 문제를 완화하기 위해 데이터 간 단위와 변수명을 프로젝트 표준 스키마로 단일화함.
- 이질적 데이터 소스 간 정합성을 확보해 통합 학습 데이터 구축 기반을 마련함.

### 2.2 검증 신뢰도 제고를 위한 데이터 분할 고도화
- 동일 제품군 내 측정값 중복으로 발생 가능한 정보 누출을 차단하기 위해 제품군 기반 그룹 분할 알고리즘 설계를 고도화함.
- 장재훈 교수님 피드백을 반영해 학습 시 관측하지 않은 신규 제품군에 대한 일반화 성능 검증 타당성을 강화함.
- 임의 샘플 분할 대비 실전 환경 유사도가 높은 검증 체계로 전환 기반을 확보함.

### 2.3 도메인 지식 기반 특성 공학 심화
- 타겟 변수의 넓은 스케일을 안정적으로 다루기 위해 로그 변환 민감도를 분석하고, 역변환 시 발생 가능한 물리적 오차 범위를 시뮬레이션함.
- 재료공학 관점에서 수명 영향력이 낮은 변수를 식별하고, 동일 성능 대비 해석력을 높이기 위한 차원 축소 및 특성 선택 전략을 구체화함.
- 성능과 해석 가능성의 균형을 맞추는 피처셋 재구성 방향을 도출함.

## 3. 주요 의사결정 사항
- 통합 데이터셋 구축을 위해 실험 데이터와 외부 크롤링 데이터의 스키마 단일화 원칙을 유지함.
- 검증 전략은 제품군 기반 그룹 분할을 기본 프레임으로 채택해 정보 누출 가능성을 최소화함.
- 타겟 로그 변환은 민감도와 역변환 오차를 함께 평가하는 방식으로 운영함.
- 저기여 변수 제거와 차원 축소를 병행하는 특성 최적화 방안을 다음 단계 기본안으로 확정함.

## 4. 리스크 및 대응
- 리스크 1: 이종 데이터 병합 과정에서 단위/정의 불일치로 인한 품질 저하
  - 대응: 표준 스키마 기준 매핑 테이블 관리, 변수 정의 검증 체크리스트 운영
- 리스크 2: 제품군 분포 편차로 인한 검증 결과 불안정성
  - 대응: 제품군 기반 그룹 분할 고정, 군별 성능 지표 병행 모니터링
- 리스크 3: 로그 역변환 시 물리적 해석 오차 확대 가능성
  - 대응: 민감도 시뮬레이션 기반 허용 오차 범위 설정 및 보정 로직 검토

## 5. 차주 계획
- 신규 병합 데이터에 확정된 통합 전처리 로직을 일괄 적용해 최종 학습용 데이터셋을 구축함.
- 제품군 단위로 재구성된 데이터셋 기반으로 모델 벤치마킹을 수행하고 성능을 재측정함.
- 라르손-밀러 파라미터(LMP) 등 물리 기반 피처의 유효성을 최종 검증함.

## 6. 기대 산출물
- 통합 스키마 매핑 정의서 및 전처리 반영 최종 학습 데이터셋
- 제품군 기반 분할 검증 결과표 및 모델 벤치마킹 리포트
- 로그 변환 민감도/역변환 오차 분석 메모와 물리 기반 피처 유효성 검증 결과

## 7. 결론
- 금주에는 데이터 소스 통합과 제품군 기반 검증 설계 고도화를 통해 모델 학습·검증 체계의 실전 적합성을 높였음.
- 다음 단계에서는 통합 전처리 파이프라인을 본격 가동하고, 재구성된 데이터셋에서 성능과 물리 기반 피처의 실효성을 정량적으로 확정하는 것이 핵심임.
