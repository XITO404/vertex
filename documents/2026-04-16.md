# 회의록 (2026-04-16)

## 1. 회의 개요
- 프로젝트명: 소재부품 수명 예측 모델 개발
- 회의 목적: 가혹도 지수 정의 전환(LMP)과 데이터 통합 파이프라인 구축 현황 점검, 커스텀 모델 설계 방향 확정

## 2. 금주 진행 사항

### 2.1 라르손-밀러 파라미터 기반 가혹도 지수 피처화
- 열처리 공정 가혹도 지수 산출 기준을 기존 검토 대상이던 Hollomon-Jaffe Parameter에서 Larson-Miller Parameter(LMP) 중심으로 전환함.
- 고온 환경 파단 시간 예측에 표준적으로 활용되는 LMP를 가혹도 지수로 채택하고 학습 데이터 핵심 피처로 반영함.
- 물리 기반 지수의 도메인 적합도를 높여 수명 예측의 해석 가능성과 실효성을 강화함.

### 2.2 학습 데이터셋 확장 및 통합 파이프라인 구축
- 모델 예측 신뢰도 향상을 위해 외부 데이터셋을 추가 확보해 학습 데이터 규모를 확장함.
- 기존 데이터와 신규 데이터 간 단위 통일, 스케일링, 결측치 처리 규칙을 포함한 통합 전처리 파이프라인 코드 구축을 완료함.
- 이종 소스 병합 이후에도 일관된 데이터 품질을 유지할 수 있는 운영 기반을 마련함.

### 2.3 예측 모델 아키텍처 수립
- 기존 범용 모델(XGBoost) 단독 활용을 넘어 프로젝트 특수성을 반영한 모델 설계에 착수함.
- Transformer와 트리 기반 모델을 결합한 앙상블 아키텍처를 설계해 변수 간 관계 학습 능력과 정형 수치 데이터 예측 강점을 동시 확보하는 방향으로 구체화함.
- 향후 재구성 데이터셋 기반 학습을 위한 커스텀 모델 구현 로드맵을 정리함.

## 3. 주요 의사결정 사항
- 가혹도 지수는 LMP를 중심으로 운영하고, 해당 지수를 핵심 피처셋에 포함함.
- 통합 데이터 전처리 파이프라인(단위 통일, 스케일링, 결측치 처리)을 표준 학습 데이터 생성 경로로 확정함.
- 모델 전략은 범용 단일 모델보다 Transformer-트리 결합형 커스텀 앙상블로 고도화함.
- 차주에는 아키텍처 구현과 제품군 단위 벤치마킹을 우선 수행함.

## 4. 리스크 및 대응
- 리스크 1: 신규 아키텍처 복잡도 상승에 따른 학습/운영 비용 증가
  - 대응: 단계적 구현(베이스라인-확장), 모듈별 성능 기여도 검증 후 채택
- 리스크 2: 이종 데이터 통합 후 분포 차이로 인한 일반화 성능 저하
  - 대응: 통합 파이프라인 일괄 적용, 제품군 단위 검증 및 군별 오차 모니터링
- 리스크 3: LMP 피처 도입 효과의 과대평가 가능성
  - 대응: 물리 기반 변수 반응 분석과 제거 실험(ablation)으로 기여도 정량 검증

## 5. 차주 계획
- 커스텀 모델 아키텍처를 코드로 구현하고 재구성 데이터셋 기반 학습을 진행함.
- 제품군 단위 데이터셋으로 신규 모델 예측 정확도를 벤치마킹하고 성능을 비교 검증함.
- 물리 기반 변수(LMP 등)에 대한 모델 반응을 분석해 판단 로직의 타당성을 점검함.

## 6. 기대 산출물
- LMP 기반 가혹도 지수 반영 피처셋 정의서
- 통합 전처리 파이프라인 적용 완료 학습 데이터셋
- 커스텀 앙상블 모델 구현 코드 및 제품군 단위 벤치마킹 결과표
- 물리 기반 변수 반응 분석 메모(기여도/해석 결과 포함)

## 7. 결론
- 금주에는 가혹도 지수 기준을 LMP로 전환하고 데이터 통합 파이프라인을 구축해 학습 데이터 신뢰 기반을 강화했으며, 커스텀 모델 설계 방향을 구체화함.
- 다음 단계의 핵심은 설계한 아키텍처를 구현해 제품군 단위 성능을 검증하고, 물리 기반 변수의 실제 기여도를 정량적으로 확인하는 것임.
