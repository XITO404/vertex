# 회의록 (2026-04-25)

## 1. 회의 개요
- 프로젝트명: 소재부품 수명 예측 모델 개발
- 회의 목적: 커스텀 앙상블 모델 설계 및 구현 현황 점검, 물리 기반 변수 반응 검증 결과 공유, 차주 고도화 계획 확정

## 2. 금주 진행 사항

### 2.1 커스텀 앙상블 모델 설계 및 1차 구현
- Transformer 인코더와 트리 기반 모델의 결합 목적 및 역할 분담을 포함한 커스텀 아키텍처 설계안을 수립함.
- 입력 피처 처리 흐름(물리 기반 피처 임베딩 -> 상호작용 학습 -> 트리 기반 보정)과 최종 예측 결합 방식을 정의함.
- 변수 간 학습 블록과 정형 수치 예측 블록을 결합해 수명 예측 성능을 동시에 확보할 수 있는 구조를 적용함.

### 2.2 물리 기반 변수 반응 및 해석 가능성 검증
- 라르손-밀러 파라미터(LMP)와 가혹도 지수의 변수 기여도 분석을 수행해 모델 판단 근거를 점검함.
- 고온, 장시간 조건에서 수명 예측 감소 경향이 물리적 상식과 일치함을 확인해 피처 유효성을 재검증함.
- 물리 기반 변수 반응 방향성이 설계 의도와 부합함을 확인해 해석 가능성 확보 수준을 점검함.

## 3. 주요 의사결정 사항
- 커스텀 모델은 Transformer-트리 결합형 앙상블 구조를 기본 아키텍처로 유지함.
- 입력 처리 흐름은 물리 기반 피처 중심으로 고정하고, 차주에는 결합 가중치 및 하이퍼파라미터를 최적화함.
- 성능 평가는 신규 제품군 중심 반복 검증과 불확실성 지표를 함께 관리하는 방식으로 확정함.

## 4. 리스크 및 대응
- 리스크 1: 모델 구조 복잡도 증가로 학습 시간 및 운영 비용이 상승할 수 있음
  - 대응: 파라미터 탐색 자동화와 경량 설정 후보를 병행 검증해 성능-비용 균형점을 도출함.
- 리스크 2: 제품군별 데이터 편차로 일반화 성능 변동 폭이 확대될 수 있음
  - 대응: 신규 제품군 중심 반복 검증과 군별 오차 추적 지표를 운영해 성능 안정성을 관리함.
- 리스크 3: 극단 조건에서 예측 불확실성이 커질 수 있음
  - 대응: 신뢰구간 산출 및 고오차 케이스 재분석 루프를 추가해 예측 결과 해석 신뢰도를 높임.

## 5. 차주 계획
- Transformer 블록 크기, attention head 수, 트리 모델 깊이 등 핵심 파라미터를 수정 보완함.
- 성능과 계산 비용을 함께 고려해 최종 운영 모델 구조를 확정함.
- 신규 제품군 중심 반복 검증으로 일반화 성능 변동 폭을 점검함.
- 예측 구간(신뢰구간) 산출 로직을 추가해 모델 결과 해석 신뢰도를 높임.

## 6. 기대 산출물
- 커스텀 앙상블 아키텍처 상세 설계 문서(입력 처리 흐름 및 결합 방식 포함)
- 핵심 파라미터 탐색 결과표와 최종 운영 모델 후보안
- 신규 제품군 기반 일반화 성능 검증 리포트
- 예측 신뢰구간 산출 로직 및 불확실성 분석 메모

## 7. 결론
- 금주에는 커스텀 앙상블 모델의 설계와 1차 구현을 완료하고, 물리 기반 변수 반응 검증을 통해 모델 해석 가능성을 점검함.
- 차주 핵심은 아키텍처 고도화와 불확실성 정량화를 통해 운영 가능한 수준의 성능 안정성과 신뢰도를 확보하는 것임.
